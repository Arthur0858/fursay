`/Users/mac/Documents/New project 3/fursay/project-control-vault`

`project-control-vault`

## Windows Setup

On Windows, open the copied or cloned `project-control-vault/` folder as the vault.

Read in this order:

1. [[00 Home]]
2. [[Dashboard]]
3. [[Handoff for Claude]]
4. [[Render Queue]]

## Daily Notes

Daily notes live in:

`Daily Notes/`

Use the template:

`Templates/Daily Note.md`

Record only operational status, changes, blockers, render queue notes, and next steps.

## Templates

- `Templates/Daily Note.md`
- `Templates/Render Job.md`
- `Templates/Completion Note.md`

## Sync Rules

- Markdown notes are safe to sync.
- Obsidian app settings in `.obsidian/` are local-only.
- API keys, `.env`, generated videos, model caches, and machine paths do not go into Obsidian notes.
- Use relative paths for project files whenever possible.

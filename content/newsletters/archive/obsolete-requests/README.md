# Obsolete Newsletter Requests

This folder keeps historical `prepare` requests that no longer represent active newsletter work.

The archived requests here were moved out of `content/newsletters/pending` because their required newsletter JSON files were never created, and later runs already produced usable newsletter artifacts for the same channel and episode. They are retained for audit history only.

Active newsletter requests must stay in `content/newsletters/pending` and must have their matching `.newsletter.json` before the release gate passes.
